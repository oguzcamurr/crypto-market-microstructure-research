# Crypto Market Microstructure Research

This project explores short-term directional prediction using top-of-book cryptocurrency market data.

The research focuses on:

* order book imbalance
* spread behavior
* microprice deviation
* short-term volatility
* large-move directional classification
* fee-adjusted expectancy evaluation

A machine learning pipeline was developed using engineered microstructure features and large-move filtering to estimate future directional price movements under realistic transaction cost assumptions.

## Features

* Spread and spread percentage
* Order book imbalance
* Microprice deviation
* Volatility estimation
* Imbalance momentum
* Large-move filtering framework
* Fee-adjusted return evaluation

## Models

* Logistic Regression (baseline)
* Random Forest Classifier

## Notes

This project was developed as part of a quantitative research case study focused on market microstructure analysis, latency-aware estimation methods, and realistic evaluation workflows.
